# Plan: Add Export Option to Livewire Report Components

## Objective
Add an "Export" button alongside the existing "Print" button in three Livewire report components to export table data (Excel/CSV) reflecting current filters and visible columns.

## Components to Modify
1. `resources/views/livewire/admin/reports/trips-report.blade.php`
2. `resources/views/livewire/admin/reports/trucks-report.blade.php`
3. `resources/views/livewire/admin/reports/drivers-report.blade.php`
4. Corresponding Livewire component classes:
   - `app/Livewire/Admin/Reports/TripsReport.php`
   - `app/Livewire/Admin/Reports/TrucksReport.php`
   - `app/Livewire/Admin/Reports/DriversReport.php`

## Prerequisites
- Laravel Excel package (`maatwebsite/excel`) must be installed. If not present, run:
  ```bash
  composer require maatwebsite/excel
  ```

## Changes Details

### Blade Files (View)
For each Blade file:
- Locate the existing Print button within the `<div class="d-flex align-items-center gap-2 flex-wrap no-print">` container.
- Add an Export button after the Print button with similar styling:
  ```html
  <button wire:click="exportReport" class="btn btn-outline-success btn-sm">
      <i class="bi bi-file-earmark-excel me-1"></i>Export
  </button>
  ```
- Ensure the button uses the same `no-print` class so it's hidden during print.

### Livewire Components
For each component class:
1. Add use statement for Excel facade (if not already present):
   ```php
   use Maatwebsite\Excel\Facades\Excel;
   ```
2. Add an `exportReport` method that:
   - Gathers the table data to be exported (same data used in the bottom table of the report).
   - Formats data as an array with appropriate headers.
   - Returns an Excel download response using `Excel::download(new ExportCollection($data, $headers), 'filename.xlsx')`.
   - Alternatively, use Excel::store or Excel::download with a callback for simplicity.
3. Since each report has different data structure, create separate export methods.

#### Specific Data to Export
- **TripsReport**: Export the `trip_rows` from `$this->summary` with columns:
  #, Date, Party, Truck, Driver, Origin, Destination, Freight (₹), Expenses (₹), Profit (₹), Status
- **TrucksReport**: Export the `truck_performance` collection from `$this->summary` with columns:
  Truck Number, Type, Assigned Driver, Trips Count, Income, Expenses, Maintenance, Profit, Utilization %
- **DriversReport**: Export the `driver_performance` collection from `$this->summary` with columns:
  Driver Name, Mobile, Truck Assigned, Trips Count, Earnings, Avg. Earnings/Trip, Completed Trips, Ongoing Trips

#### Export Method Example (TripsReport)
```php
public function exportReport()
{
    $data = [];
    foreach ($this->summary['trip_rows'] as $index => $trip) {
        $data[] = [
            '#' => $index + 1,
            'Date' => $trip['date'],
            'Party' => $trip['party'],
            'Truck' => $trip['truck'],
            'Driver' => $trip['driver'],
            'Origin' => $trip['origin'],
            'Destination' => $trip['destination'],
            'Freight (₹)' => $trip['freight_amount'],
            'Expenses (₹)' => $trip['expenses'],
            'Profit (₹)' => $trip['profit'],
            'Status' => ucwords(str_replace('_', ' ', $trip['status'])),
        ];
    }

    return Excel::download(new \Maatwebsite\Excel\Concerns\FromArray($data), 'trips-report.xlsx');
}
```
*Note: For production, create proper Export classes implementing `FromCollection` or `WithHeadings`. For simplicity, we can use `FromArray` with headings added as first row.*

Better approach: Use `WithHeadings` and `FromCollection`:
```php
use Maatwebsite\Excel\Concerns\FromCollection;
use Maatwebsite\Excel\Concerns\WithHeadings;

class TripsExport implements FromCollection, WithHeadings
{
    public function __protected $data;

    public function __construct($data)
    {
        $this->data = $data;
    }

    public function collection()
    {
        return collect($this->data);
    }

    public function headings(): array
    {
        return [
            '#', 'Date', 'Party', 'Truck', 'Driver', 'Origin', 'Destination',
            'Freight (₹)', 'Expenses (₹)', 'Profit (₹)', 'Status'
        ];
    }
}
```
Then in component:
```php
public function exportReport()
{
    $data = [];
    foreach ($this->summary['trip_rows'] as $index => $trip) {
        $data[] = [
            $index + 1,
            $trip['date'],
            $trip['party'],
            $trip['truck'],
            $trip['driver'],
            $trip['origin'],
            $trip['destination'],
            $trip['freight_amount'],
            $trip['expenses'],
            $trip['profit'],
            ucwords(str_replace('_', ' ', $trip['status'])),
        ];
    }

    return Excel::download(new TripsExport($data), 'trips-report.xlsx');
}
```
